## Use a predictable hierarchy

Common sections include education, experience, leadership or extracurricular activities, and skills. Put the most relevant evidence where it can be found quickly. Within a section, use a consistent order and make employer, role, location and dates unambiguous.

## Consistency is part of credibility

Choose one date format, one punctuation approach and one set of heading styles. Check spacing, alignment and page breaks in the final document. The workshop stores structured text; it does not assess fonts, margins, visual hierarchy or PDF rendering.

```callout
variant: tip
title: Follow the application instructions
md: If an employer specifies a file type, page limit or required information, that instruction takes priority over a general template.
```

## Keep editing separate from formatting

First make the content accurate and relevant. Then perform a deliberate visual pass in the final document. Separating these jobs makes it easier to notice both weak evidence and layout problems.
